136-SingleNumber

  • 136. 只出现一次的数字

    Difficulty: 简单

    给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

    说明：

    你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

    示例 1:

    输入: [2,2,1]
输出: 1

    示例 2:

    输入: [4,1,2,1,2]
输出: 4

    Solution

    注意两点：

    1. 线性时间复杂度
    2. 不使用额外空间

    那么可以使用位运算，从头异或到尾，使用nums[0]作为每次异或更新的结果，最后返回即可

    class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        for i in range(1, len(nums)):
            nums[0] ^= nums[i]
        return nums[0]

    复杂度分析

    令 n 为数组长度。

    • 时间复杂度：O(n)
    • 空间复杂度：O(1)
