229-求众数-II

229. 求众数 II

Difficulty: 中等

给定一个大小为 n 的整数数组，找出其中所有出现超过 ⌊ n/3 ⌋ 次的元素。

  • 进阶：*尝试设计时间复杂度为 O(n)、空间复杂度为 O(1)的算法解决此问题。

示例 1：

输入：[3,2,3]
输出：[3]

示例 2：

输入：nums = [1]
输出：[1]

示例 3：

输入：[1,1,1,3,3,2,2,2]
输出：[1,2]

提示：

  • 1 <= nums.length <= 5 * 10<sup>4</sup>
  • 10<sup>9</sup> <= nums[i] <= 10<sup>9</sup>

Solution

本题考察的是对摩尔投票法的理解程度，只需在原有的主站169题的基础上稍加改动，即可完成解答。

关于摩尔投票法的理解，参考https://www.zhihu.com/question/49973163/answer/235921864

本题的思路就是，同时维护两组数值，分别是对应的major和cnt，然后两次for循环，一次抵消，一次计数，最后分情况返回即可，注意返回的是个列表

class Solution:
    def majorityElement(self, nums: List[int]) -> List[int]:
        length = len(nums)
        major1 = major2 = nums[0]
        cnt1 = cnt2 = 0
        # 维护两组数值，代表额外两个用来抵消的数字
        for i in range(length):
            if major1 == nums[i]:
                cnt1 += 1
                continue
            if major2 == nums[i]:
                cnt2 += 1
                continue
            if cnt1 == 0:
                major1 = nums[i]
                cnt1 = 1
                continue
            if cnt2 == 0:
                major2 = nums[i]
                cnt2 = 1
                continue
            cnt1 -= 1
            cnt2 -= 1
        # 重新计数。确认出现次数
        cnt1 = cnt2 = 0
        for num in nums:
            if num == major1:
                cnt1 += 1
            elif num == major2:
                cnt2 += 1
        # 分情况返回
        if cnt1 > length / 3 and cnt2 > length / 3:
            return [major1, major2]
        else:
            if cnt1 > length / 3:
                return [major1]  
            elif cnt2 > length / 3: 
                return [major2]
            else:
                return []

复杂度分析

令 n 为数组长度。

  • 时间复杂度：O(n)
  • 空间复杂度：O(1)
