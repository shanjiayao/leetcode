453-最小操作次数使数组元素相等

453. 最小操作次数使数组元素相等

Difficulty: 简单

给定一个长度为 n 的 非空 整数数组，每次操作将会使 n - 1 个元素增加 1。找出让数组所有元素相等的最小操作次数。

示例：

输入：
[1,2,3]
输出：
3
解释：
只需要3次操作（注意每次操作会增加两个元素的值）：
[1,2,3]  =>  [2,3,3]  =>  [3,4,3]  =>  [4,4,4]

Solution

读题之后应该可以发现，这道题难点不在编程，在于思想，在于是否可以捕捉到其中的规律

给定一个数组，为了更好地找规律，我们先对其进行升序排序

[1, 3, 4, 5, 6, 7] 除7之外所有元素加上 7-1

[7, 9, 11, 12, 13, 7] 除13之外，所有元素加上 13 - 7

[13, 15, 17, 18, 13, 13] 除18外，所有元素加上 18 - 13

[18, 20, 22, 18, 18, 18] 除22外，所有元素加上 22 - 18

[22, 24, 22, 22, 22, 22] 除24外，所有元素加上 24 - 22

[24, 24, 24, 24, 24, 24] 完成！

经过上述过程可以发现，利用排序之后的数组顺序关系，我们可以每次同化一个数字，一共需要数组 长度-1次，每次移动的长度等于最大数减去数组第一个元素（最小数）。此外，由于N-1个元素都是一起加1的，所以可以发现，每次移动的长度刚好等于原数组排序后，后面每个数减去第一个数的值，求和即可。

class Solution:
    def minMoves(self, nums: List[int]) -> int:
        nums.sort()
        steps = 0
        for i in nums:
            steps += i - nums[0]
        return steps

复杂度分析

令 n 为数组长度。

  • 时间复杂度：O(nlogn) 排序
  • 空间复杂度：O(n)
