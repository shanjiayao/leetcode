17-10-主要元素

面试题 17.10. 主要元素

Difficulty: 简单

数组中占比超过一半的元素称之为主要元素。给定一个整数数组，找到它的主要元素。若没有，返回-1。

示例 1：

输入：[1,2,5,9,5,9,5,5,5]
输出：5

示例 2：

输入：[3,2]
输出：-1

示例 3：

输入：[2,2,1,1,1,2,2]
输出：2

说明： 你有办法在时间复杂度为 O(N)，空间复杂度为 O(1) 内完成吗？

Solution

看到题，第一反应就是哈希表存一下次数，然后判断最大次数的元素是否满足。但是奈何时间限制，所以就只能用摩尔投票法

关于摩尔投票法的原理不再赘述，这里需要注意的是，摩尔投票之后，，还需要再遍历一次

代码如下

class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        major, cnt = nums[0], 1
        # 第一次遍历，找到主要元素
        for i in range(1, len(nums)):
            if cnt == 0:
                major = nums[i]
                cnt = 1
            else:
                cnt = cnt + 1 if major == nums[i] else cnt - 1
        # 由于找到的major可能是[1, 2, 3]的情况，所以需要再遍历一次，确定出现次数，去伪存真
        cnt = 0
        for num in nums:
            if num == major:
                cnt += 1

        return major if cnt > len(nums) / 2 else -

复杂度分析

令 n 为数组长度。

  • 时间复杂度：O(n)
  • 空间复杂度：O(1)
