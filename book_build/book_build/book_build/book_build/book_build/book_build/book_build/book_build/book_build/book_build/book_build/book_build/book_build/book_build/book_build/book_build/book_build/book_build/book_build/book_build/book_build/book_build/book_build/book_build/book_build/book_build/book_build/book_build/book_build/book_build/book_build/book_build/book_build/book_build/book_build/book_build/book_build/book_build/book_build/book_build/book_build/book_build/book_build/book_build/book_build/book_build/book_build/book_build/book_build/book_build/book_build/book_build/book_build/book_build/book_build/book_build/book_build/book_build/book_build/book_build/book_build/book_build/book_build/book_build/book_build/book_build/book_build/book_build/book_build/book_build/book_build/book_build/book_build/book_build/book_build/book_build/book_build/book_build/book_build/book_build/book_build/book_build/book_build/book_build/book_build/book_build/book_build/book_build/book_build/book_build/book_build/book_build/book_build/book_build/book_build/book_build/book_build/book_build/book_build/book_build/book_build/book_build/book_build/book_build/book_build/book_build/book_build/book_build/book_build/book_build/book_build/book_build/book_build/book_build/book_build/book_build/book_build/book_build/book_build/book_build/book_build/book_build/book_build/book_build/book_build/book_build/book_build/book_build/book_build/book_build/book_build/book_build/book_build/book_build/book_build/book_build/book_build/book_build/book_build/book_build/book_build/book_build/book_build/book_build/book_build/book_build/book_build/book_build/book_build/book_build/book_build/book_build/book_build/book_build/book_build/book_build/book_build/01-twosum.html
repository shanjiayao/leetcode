TwoSum

题目

1. 两数之和

Difficulty: 简单

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。

示例:

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

Solution

思路：

  • 暴力搜索

    两次遍历，查看列表中的元素之和，判断是否等于target

    class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        for i in range(len(nums)):
                        # 注意这里的遍历范围，是从 i+1开始，到len（nums）截止，步长为1
            for j in range(i + 1, len(nums), 1):
                if nums[i] + nums[j] == target:
                    return [i, j]
        return None

    时间复杂度：O(n^2)

    空间复杂度：O(1)

  • 哈希表：

    空间换时间

    先遍历一遍，将给定的列表存成哈希表，在python里面可以用字典表示，键为列表每个元素的值，键值就是每个元素的索引

    class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        table = {}
        for i in range(len(nums)):
            table.update({
                nums[i]: i
            })
        for i in range(len(nums)):
            if target - nums[i] in table.keys():
                if table[target - nums[i]] != i:
                    return [i, table[target - nums[i]]]
        return None

    时间复杂度：O(n)

    空间复杂度：O(n)
