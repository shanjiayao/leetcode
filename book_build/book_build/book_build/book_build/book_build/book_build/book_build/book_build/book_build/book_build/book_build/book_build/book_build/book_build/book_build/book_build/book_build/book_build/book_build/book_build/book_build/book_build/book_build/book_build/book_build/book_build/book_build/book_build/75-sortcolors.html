75-SortColor

75. 颜色分类

Difficulty: 中等

给定一个包含红色、白色和蓝色，一共 n个元素的数组，对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

示例 1：

输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]

示例 2：

输入：nums = [2,0,1]
输出：[0,1,2]

示例 3：

输入：nums = [0]
输出：[0]

示例 4：

输入：nums = [1]
输出：[1]

提示：

  • n == nums.length
  • 1 <= n <= 300
  • nums[i] 为 0、1 或 2

进阶：

  • 你可以不使用代码库中的排序函数来解决这道题吗？
  • 你能想出一个仅使用常数空间的一趟扫描算法吗？

Solution

题意分析：

  • 其实就是在排序
  • 不适用库函数
  • 空间复杂度为O(1)

这里可以看一下常见的排序算法对应的时空复杂度

  1. 选择排序
    1. 直接选择排序
    2. 堆排序
  2. 插入排序
    1. 直接插入排序
    2. 希尔排序
  3. 交换排序
    1. 冒泡排序
    2. 快速排序
  4. 归并排序

对应的时空复杂度如下：

下面简单实现下冒泡排序的方法

class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        length = len(nums) - 1
        for i in range(length):
            for j in range(length):
                k = nums[j]
                if nums[j] > nums[j+1]:
                    nums[j] = nums[j+1]
                    nums[j+1] = k

时间复杂度：O(n^2)

空间复杂度：O(1)
