448-找到所有数组中消失的数字

448. 找到所有数组中消失的数字

Difficulty: 简单

给定一个范围在 1 ≤ a[i] ≤ n ( n = 数组大小 ) 的 整型数组，数组中的元素一些出现了两次，另一些只出现一次。

找到所有在 [1, n] 范围之间没有出现在数组中的数字。

您能在不使用额外空间且时间复杂度为O(n)的情况下完成这个任务吗? 你可以假定返回的数组不算在额外空间内。

示例:

输入:
[4,3,2,7,8,2,3,1]

输出:
[5,6]

Solution

既然要求了不能用额外空间，并且线性的时间复杂度，那么排序的方法不能用，位运算的话，可以找到所有的丢失数字异或总值，但是不能确定一共多少丢失的数字，也不适合

那么参考官方题解的思路，对原数组进行原地操作，以达到无额外空间复杂度

为了区分存在的元素以及丢失的元素，遍历一遍数组的同时，把每个元素x对应位置上的值加上一个定值，这个值需大于数组长度，这样就和原数组有了明显的区别

最后再遍历一次，如果某个位置上的元素小于数组长度，那么就说明丢失的数字是这个位置索引+1

class Solution:
    def findDisappearedNumbers(self, nums: List[int]) -> List[int]:
        length = len(nums)
        for i in nums:
            x = (i - 1) % length
            nums[x] += length

        return [ i + 1 for i in range(length) if nums[i] <= length ]

复杂度分析

令 n 为数组长度。

  • 时间复杂度：O(n)
  • 空间复杂度：O(1)
