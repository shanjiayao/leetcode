414-第三大的数

414. 第三大的数

Difficulty: 简单

给你一个非空数组，返回此数组中 第三大的数 。如果不存在，则返回数组中最大的数。

示例 1：

输入：[3, 2, 1]
输出：1
解释：第三大的数是 1 。

示例 2：

输入：[1, 2]
输出：2
解释：第三大的数不存在, 所以返回最大的数 2 。

示例 3：

输入：[2, 2, 3, 1]
输出：1
解释：注意，要求返回第三大的数，是指第三大且唯一出现的数。
存在两个值为2的数，它们都排第二。

提示：

  • 1 <= nums.length <= 10<sup>4</sup>
  • -2<sup>31</sup> <= nums[i] <= 2<sup>31</sup> - 1

进阶：你能设计一个时间复杂度 O(n) 的解决方案吗？

Solution

一开始考虑过使用快慢指针来做，快指针遍历所有列表中的数，慢指针处理有效的数字并且排序，不过逻辑上有点混乱

最终采用了粗暴的解法，定义前三个大的数，然后遍历数组，不断更新三个数的值，最终判断是否存在第三大的数

class Solution:
    def thirdMax(self, nums: List[int]) -> int:
        one, two, three = float('-inf'), float('-inf'), float('-inf')
        for i in nums:
            if i > one:
                three = two
                two = one
                one = i
            elif i == one:
                continue
            elif i > two:
                three = two
                two = i
            elif i == two:
                continue
            elif i > three:
                three = i
            else:
                continue
        print(one, two, three)
        if three > float('-inf'):
            return three
        else:
            return one

复杂度分析

令 n 为数组长度。

  • 时间复杂度：$O(n)$
  • 空间复杂度：$O(1)$
