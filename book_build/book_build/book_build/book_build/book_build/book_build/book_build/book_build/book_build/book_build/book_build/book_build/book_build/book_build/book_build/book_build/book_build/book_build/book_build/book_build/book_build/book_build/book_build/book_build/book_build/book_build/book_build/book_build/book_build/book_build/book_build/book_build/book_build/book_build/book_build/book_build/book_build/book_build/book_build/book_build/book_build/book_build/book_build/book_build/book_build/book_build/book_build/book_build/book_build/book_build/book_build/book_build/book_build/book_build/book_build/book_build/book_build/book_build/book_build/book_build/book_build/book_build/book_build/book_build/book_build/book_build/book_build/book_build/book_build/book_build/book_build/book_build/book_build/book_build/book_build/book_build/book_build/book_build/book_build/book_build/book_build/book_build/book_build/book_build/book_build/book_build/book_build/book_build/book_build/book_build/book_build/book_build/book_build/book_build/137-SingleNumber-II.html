137-SingleNumber-II

137. 只出现一次的数字 II

Difficulty: 中等

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现了三次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,3,2]
输出: 3

示例 2:

输入: [0,1,0,1,0,1,99]
输出: 99

Solution

当列表中每个元素的出现次数为奇数的时候，仅使用一次异或操作已经不能够帮助我们解决问题了，然而就算你使用两次异或，也依旧不行，所以需要额外的位运算符

官方题解的方法：

增加一个常数量，来记录是否出现两次，最后这个出现两次的常数量会帮助记录每个数字的出现次数是不是只有一次，返回出现一次的数字即可

  • 初始值 once 和twice 都为0
  • 计算once的时候，在异或操作的基础上，增加twice的约束，对twice取反，并按位与
  • 计算twice的时候同理
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        once = twice = 0
        for i in nums:
            once = ~twice & (once ^ i)
            twice = ~once & (twice ^ i)
        return once

复杂度分析

令 n 为数组长度。

  • 时间复杂度：O(n)
  • 空间复杂度：O(1)
