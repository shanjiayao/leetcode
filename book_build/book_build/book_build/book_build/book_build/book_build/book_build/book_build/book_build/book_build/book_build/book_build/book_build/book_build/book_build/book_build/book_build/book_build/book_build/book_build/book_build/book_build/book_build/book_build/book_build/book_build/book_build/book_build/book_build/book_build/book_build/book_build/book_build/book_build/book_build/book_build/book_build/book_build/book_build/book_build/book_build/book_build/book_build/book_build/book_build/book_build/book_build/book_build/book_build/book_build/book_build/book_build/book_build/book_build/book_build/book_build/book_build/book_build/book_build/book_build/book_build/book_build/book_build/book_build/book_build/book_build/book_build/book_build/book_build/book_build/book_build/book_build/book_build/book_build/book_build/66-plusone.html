66-加一

66. 加一

Difficulty: 简单

给定一个由 整数 组成的 非空 数组所表示的非负整数，在该数的基础上加一。

最高位数字存放在数组的首位， 数组中每个元素只存储单个数字。

你可以假设除了整数 0 之外，这个整数不会以零开头。

示例 1：

输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。

示例 2：

输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：输入数组表示数字 4321。

示例 3：

输入：digits = [0]
输出：[1]

提示：

  • 1 <= digits.length <= 100
  • 0 <= digits[i] <= 9

Solution

1. 递归法：

分情况讨论：

  1. 判断数组最后一位，如果小于9，那么直接最后一位加一并返回
  2. 如果最后一位等于9，那么最后一位要变成0，然后其余位可以使用递归进行处理，将除最后一位之前的所有位的数组元素进行递归，递归的终止条件是最后一位小于9或者最后一位等于9并且当前传入的数组元素长度为1
    • 如果小于9的话，相当于倒数第二位加一
    • 如果等于9，还需要判断数组长度，如果长度是1，那么除了最后一位元素剩下为空，不能继续递归，直接返回[1, 0]即可

Language: python

class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        length = len(digits)
        end_digit = digits[-1]
        if end_digit < 9:
            digits[-1] += 1
        else:
            if length == 1:
                return [1, 0]
            digits[-1] = 0
            digits[0:-1] = self.plusOne(digits[0:-1])
        return digits

时间复杂度：最好情况下O(1)，最坏情况下O(n)

空间复杂度：O(1)

2. 转换数据类型再加一

这种思路可以有两种方式，一种是直接将列表转换为数字（list -> int -> +1 -> list），另外一种是通过字符创转换（list -> str -> int -> str -> list）

  • 将列表转换为字符串然后再转换成整数，加一之后再变换为字符串，进而转换为列表

    • 注意使用map函数可以将列表中的每个元素转换成对应类型

    • 注意输入如果是[0, 0]，那么输出应该是[0, 1]，所以最后加了个while补全0，或者使用字符串自带的zfill函数，可以自动补0到指定长度

      class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        digits_ = map(str, digits)
        digits_str = ''.join(digits_)
        digits_str = str(int(digits_str) + 1)
        # 使用 zfill 补0
        digits_str = digits_str.zfill(len(digits))
        ret_list = []
        for x in digits_str:
            ret_list.append(int(x)) 
        return ret_list

      时间复杂度：O(n)

      空间复杂度：O(n)
