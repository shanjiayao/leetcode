59. 螺旋矩阵 II

Difficulty: 中等

给你一个正整数 n ，生成一个包含 1 到 n<sup>2</sup> 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

示例 1：

输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]

示例 2：

输入：n = 1
输出：[[1]]

提示：

  • 1 <= n <= 20

Solution

思路与上一题相同，只不过这次是向矩阵中填数字，注意几点

  • python初始化二维数组时，不要用 a = [[None]*n]*n 的方式，这样相当于把其列表做了浅拷贝，正确的应该用 for i in range 来重复
  • 这道题同样需要记录走过的路线，所以在初始化矩阵时，我将其元素均初始化为字符串，便于区分
  • 注意特殊条件的判断，本题 n大于等于1，所以不存在特殊情况
class Solution:
    def move_right(self, mat, next_val, i, j, n):
        if j < n-1:
            if isinstance(mat[i][j+1], str):
                mat[i][j+1] = next_val
                return True, i, j+1
            else: 
                return False, i, j
        else:
            return False, i, j

    def move_down(self, mat, next_val, i, j, n):
        if i < n-1:
            if isinstance(mat[i+1][j], str):
                mat[i+1][j] = next_val
                return True, i+1, j
            else: 
                return False, i, j
        else:
            return False, i, j

    def move_left(self, mat, next_val, i, j, n):
        if j > 0:
            if isinstance(mat[i][j-1], str):
                mat[i][j-1] = next_val
                return True, i, j-1
            else: 
                return False, i, j
        else:
            return False, i, j

    def move_up(self, mat, next_val, i, j, n):
        if i > 0:
            if isinstance(mat[i-1][j], str):
                mat[i-1][j] = next_val
                return True, i-1, j
            else: 
                return False, i, j
        else:
            return False, i, j

    def generateMatrix(self, n: int) -> List[List[int]]:
        ops_list = ['right', 'down', 'left', 'up']
        total_steps = n*n
        cur_step = 1
        ret_list = [['1']*n for i in range(n)]
        i = j = ops_index = 0

        ret_list[0][0] = 1
        while cur_step < total_steps:
            cur_method = getattr(self, 'move_%s' % ops_list[ops_index%4])
            state, new_i, new_j = cur_method(ret_list, cur_step+1, i, j, n)
            if state:
                cur_step += 1
            else:
                ops_index += 1

            i, j = new_i, new_j
            print(i, j)
            print(cur_step, ops_index)
            print(ret_list)
            print('----------------')
        return ret_list

复杂度分析

令n为矩阵边长

时间复杂度：O(n^2)，一个一个填入，所以时间复杂度等于矩阵元素个数

空间复杂度：O(1)，不考虑返回矩阵的空间占用
