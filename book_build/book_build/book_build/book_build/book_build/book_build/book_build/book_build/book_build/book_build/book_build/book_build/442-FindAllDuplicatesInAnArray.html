442-数组中重复的数

442. 数组中重复的数据

Difficulty: 中等

给定一个整数数组 a，其中1 ≤ a[i] ≤ n （n为数组长度）, 其中有些元素出现两次而其他元素出现一次。

找到所有出现两次的元素。

你可以不用到任何额外空间并在O(n)时间复杂度内解决这个问题吗？

示例：

输入:
[4,3,2,7,8,2,3,1]

输出:
[2,3]

Solution

与448一样，这道题只不过是在448的基础上，需要我们额外判断出现两次的元素，因为题中给出了同样的时间和空间要求，所以我们可以在448的题解基础上，稍作改动，判断丢失元素的同时，再区分出现两次的元素和出现一次的元素，这里使用的是乘了个负号，代码如下

对于任意元素num，由于可能去了相反数以及加了数组长度，所以需要先计算绝对值，减1后再对数组长度取余，这样就得到了原始元素对应的索引x，然后将索引对应的元素加上数组长度（注意这里的符号），然后再乘以-1，这样的话，相当于原来的数组会变成以下情况

  • 出现过的元素，绝对值都大于length
  • 出现一次的元素为负数
  • 出现两次的元素为正数
  • 丢失的元素小于length

稍作判断即可得到出现两次的元素

（同理，也可以得到出现一次的元素）

class Solution:
    def findDuplicates(self, nums: List[int]) -> List[int]:
        length = len(nums)
        for num in nums:
            x = (abs(num) - 1) % length
            nums[x] = nums[x] + length if nums[x] > 0 else nums[x] - length
            nums[x] *= -1
        return [i+1 for i in range(length) if nums[i] > length]

复杂度分析

令 n 为数组长度。

  • 时间复杂度：O(n)
  • 空间复杂度：O(1)
