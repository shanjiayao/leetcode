27-移出元素

27. 移除元素

Difficulty: 简单

给你一个数组 nums和一个值 val，你需要 移除所有数值等于 val的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

说明:

为什么返回数值是整数，但输出的答案是数组呢?

请注意，输入数组是以「引用」方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

你可以想象内部操作如下:

// nums 是以“引用”方式传递的。也就是说，不对实参作任何拷贝
int len = removeElement(nums, val);

// 在函数里修改输入数组对于调用者是可见的。
// 根据你的函数返回的长度, 它会打印出数组中 该长度范围内 的所有元素。
for (int i = 0; i < len; i++) {
    print(nums[i]);
}

示例 1：

输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2]
解释：函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。你不需要考虑数组中超出新长度后面的元素。例如，函数返回的新长度为 2 ，而 nums = [2,2,3,3] 或 nums = [2,2,0,0]，也会被视作正确答案。

示例 2：

输入：nums = [0,1,2,2,3,0,4,2], val = 2
输出：5, nums = [0,1,4,0,3]
解释：函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。注意这五个元素可为任意顺序。你不需要考虑数组中超出新长度后面的元素。

提示：

  • 0 <= nums.length <= 100
  • 0 <= nums[i] <= 50
  • 0 <= val <= 100

Solution

  1. 暴力遍历，判断是否等于要移除的元素val，等于的话使用python的remove方法将对应元素移除掉

    注意，如果使用for循环的话，在循环中改变列表的值，每次for循环判断的话对应的列表也是改变后的，所以会产生遗漏，解决方案：

    • 如果不考虑空间复杂度，可以将原列表进行浅拷贝，将for i in nums改为for i in nums[:]
    • 放弃for循环，使用while判断
    class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        while val in nums:
            nums.remove(val)
        return len(nums)

    时间复杂度：最好是O(n)，指数组中只含有一个目标元素的情况；最坏情况下是O(n^2)，考虑到判断val是否在nums中的时间复杂度应该是O(n)

    空间复杂度：O(1)

  2. 双指针的快慢指针，快指针遍历整个列表，慢指针保留有效元素

class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        length = len(nums)
        slow_ = 0
        for fast_ in range(length):
            if nums[fast_] != val:
                nums[slow_] = nums[fast_]
                slow_ += 1
        return len(nums[0:slow_])

时间复杂度：O(n)，快指针遍历一次

空间复杂度：O(1)
