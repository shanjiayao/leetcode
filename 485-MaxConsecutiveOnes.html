485-MaxConsecutiveOnes

485. 最大连续 1 的个数

Difficulty: 简单

给定一个二进制数组， 计算其中最大连续 1 的个数。

示例：

输入：[1,1,0,1,1,1]
输出：3
解释：开头的两位和最后的三位都是连续 1 ，所以最大连续 1 的个数是 3.

提示：

  • 输入的数组只包含 0 和 1 。
  • 输入数组的长度是正整数，且不超过 10,000。

思路

一次性遍历所有数组元素，定义两个长度，分别是全局的最长长度以及当前连续长度，在每次遍历时都更新两个长度的数值，如下

  • 如果当前元素为1，连续长度+1
  • 如果当前元素为0，连续长度置0

最后，判断最长长度是否小于连续长度，若是，将连续长度赋给最长长度

代码

class Solution:
    def findMaxConsecutiveOnes(self, nums: List[int]) -> int:
        length = 0
        sub_len = 0
        for i in nums:
            if i == 1:
                sub_len += 1
            else:
                sub_len = 0
            length = sub_len if sub_len > length else length
        return length

复杂度分析

令 n 为数组长度。

  • 时间复杂度：$O(n)$
  • 空间复杂度：$O(1)$
